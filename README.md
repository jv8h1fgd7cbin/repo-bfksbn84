# Telegram Pet Owner Finder

Интеллектуальная система поиска владельцев домашних животных (собак и кошек) в Telegram-группах через пользовательский аккаунт (MTProto / Telethon) с ИИ-анализом сообщений.

## Возможности

- Мониторинг всех доступных групп, супергрупп и форумов аккаунта
- Первоначальная загрузка истории + режим реального времени
- ИИ-анализ (OpenAI) с учётом контекста всех сообщений пользователя — не по одному слову
- Категории: собаки / кошки / кошки+собаки / не определено, автоматическая переоценка при новых сообщениях
- Дедупликация по `user_id`: повторные находки обновляют запись и дописывают историю
- Суточный лимит: 100 новых уникальных пользователей в день (настраивается), повторы не считаются
- Авто-поиск и авто-вступление (опционально, `DISCOVERY_ENABLED=true`): поиск публичных групп по ключевым словам, ИИ-проверка релевантности по образцу сообщений перед вступлением, консервативные лимиты Telegram (вступлений/сутки, паузы, FloodWait)
- Чаты без доступа (приватные/по заявке): никаких авто-вступлений — очередь + уведомление администратору с просьбой вступить вручную; после вступления индексация начинается автоматически
- Админ-панель: статистика, поиск, экспорт CSV / Excel / JSON
- Автовосстановление после сбоев, обработка FloodWait, подробное логирование

## Запуск

1. Скопируйте конфиг и заполните значения:

   ```bash
   cp .env.example .env
   ```

   Нужны: `TELEGRAM_API_ID` / `TELEGRAM_API_HASH` (с https://my.telegram.org), `OPENAI_API_KEY`, пароль PostgreSQL.

2. Первый вход в аккаунт (создание сессии Telethon) — интерактивно:

   ```bash
   docker compose run --rm app python -m app.main
   ```

   Введите номер телефона и код подтверждения. Сессия сохранится в volume.

3. Запуск в фоне:

   ```bash
   docker compose up -d --build
   ```

4. Админ-панель: http://localhost:8080

## API

- `GET /api/stats` — статистика
- `GET /api/search?username=&user_id=&name=&category=&chat_id=&date_from=&date_to=` — поиск
- `GET /api/export/{dog|cat|dog_and_cat|undefined|all}/{csv|xlsx|json}` — экспорт

## Архитектура

```
app/
  config.py            # все настройки через .env
  db/                  # PostgreSQL (SQLAlchemy async)
  services/
    ai_analyzer.py     # ИИ-классификация по контексту сообщений
    daily_limit.py     # суточный лимит новых user_id (Redis)
    repository.py      # upsert/поиск/статистика
    exporter.py        # CSV / Excel / JSON
  telegram/monitor.py  # Telethon: бэкфилл, realtime, очередь недоступных чатов
  admin/               # FastAPI админ-панель
```
